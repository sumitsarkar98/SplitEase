# SplitEase 💰

**SplitEase** is a smart personal finance tracker that helps you manage expenses and budgets.

---

## Functional Requirements

### User Account Management
- Users can register, login, and logout securely
- Users can update profile details
- Users can reset passwords

---

### Dashboard
Displays a quick financial overview:

- Total Balance
- Total Income
- Total Expenses
- Recent Transactions
- Budget Progress
- Expense vs Income Chart

---

### Transaction Management
Users can manage financial transactions.

- Add new transaction
- Edit transaction
- Delete transaction
- Categorize transactions
- Add description / notes
- Add transaction date
- Mark transaction type (Income / Expense)

---

### Budget Management
- Create monthly budget
- Set category-wise budget
- Track budget progress
- Notify when budget is near limit
- Show remaining budget

---

### Categories

- Family

---

### Insights & Analytics
- Monthly spending summary
- Category-wise expense breakdown
- Income vs expense comparison
- Budget usage insights
- Spending trend over time

---

### Notifications
Delivery type: **In-app notifications**

- Budget exceeded
- Budget nearing limit
- Monthly summary
- Reminder to add transactions

---

### Search & Filters
Users can easily find transactions using:

- Filter by date
- Filter by category
- Filter by amount
- Filter by transaction type
- Search by keyword
